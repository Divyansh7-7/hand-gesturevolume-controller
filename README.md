# 🔊 Volume Controller

A lightweight and user-friendly **Volume Controller** application that allows users to adjust the system volume seamlessly using an intuitive interface. The project is designed to demonstrate practical use of system automation and user interaction in Python.

---

## 📖 Overview

Volume Controller is a desktop utility that enables users to control their computer's audio output efficiently. It provides a simple interface for increasing, decreasing, muting, or setting the system volume without navigating through operating system settings.

This project is ideal for learning desktop automation, Python programming, and system-level interactions.

## 🚀 Installation

### Clone the repository

```bash
git clone https://github.com/Divyansh7-7/volume-controller.git
```

Move into the project directory

```bash
cd volume-controller
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the project

```bash
python main.py
```


## ⚙️ How It Works

1. Launch the application.
2. The program connects with the Windows Audio API.
3. User interactions are captured.
4. Volume level is adjusted in real time.
5. Changes are immediately reflected in the system.
